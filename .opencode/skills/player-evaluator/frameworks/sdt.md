# 自我决定理论 (Self-Determination Theory)

基于 Edward Deci 和 Richard Ryan 的动机心理学研究。

---

## 三种基本心理需求

### 1. 自主性 (Autonomy)

**定义**: 感觉行为是出于自己的选择，而非外部压力。

**游戏应用**:
- **有意义的选择**: 决策影响结果
- **多路径玩法**: 不同方式达成目标
- **自定义选项**: 角色外观、技能配置
- **非强制引导**: 玩家可探索

**设计检查**:
- [ ] 玩家是否有真实选择?
- [ ] 路径是否多样?
- [ ] 是否过度引导?

**自主性破坏**:
- 只有一条正确路径
- 强制教程过长
- 无法自定义
- 随机决定结果

---

### 2. 胜任感 (Competence)

**定义**: 感觉有能力应对挑战，能够成长和精通。

**游戏应用**:
- **技能成长**: 可见的进步
- **挑战匹配**: 难度与能力平衡
- **精通系统**: 深度机制
- **成就系统**: 认可玩家能力

**设计检查**:
- [ ] 进步是否可见?
- [ ] 挑战是否公平?
- [ ] 是否有精通空间?
- [ ] 成就是否代表真实技能?

**胜任感破坏**:
- 随机成功（无技能）
- 难度过低（无挑战）
- 无进度反馈
- 难度过高（无成长）

---

### 3. 关系感 (Relatedness)

**定义**: 感觉与他人有连接，属于某个社群。

**游戏应用**:
- **社交功能**: 好友、聊天、公会
- **合作内容**: 团队任务、共享目标
- **社区认同**: 专属称号、徽章
- **NPC 关系**: 有意义的角色互动

**设计检查**:
- [ ] 是否有社交功能?
- [ ] 合作内容是否有意义?
- [ ] 社区工具是否完善?
- [ ] NPC 是否有深度?

**关系感破坏**:
- 强制 PvP（有害社交）
- 有毒社区无管理
- 无社交工具
- NPC 扁平化

## 动机类型

### 内在动机 vs 外在动机

| 类型 | 来源 | 特征 | 持久性 |
|------|------|------|--------|
| **内在** | 活动本身 | 享受、兴趣、满足 | 长期 |
| **外在** | 外部奖励 | 奖励、惩罚、认可 | 短期 |

**设计目标**: 培养内在动机，外在奖励作为辅助。

### 动机光谱

```
无动机 ─ 外在动机 ─ 内化动机 ─ 内在动机
  ↓         ↓          ↓         ↓
不关心    被迫      认同价值    享受本身
```

**外在动机层次**:
1. 外部调节 (仅奖励/惩罚)
2. 内射调节 (内疚/自豪)
3. 认同调节 (认同价值)
4. 整合调节 (与自我一致)

**游戏设计**: 奖励应该支持内化，而非替代内在动机。

## 游戏设计应用

### 自主性设计模式

| 模式 | 描述 | 示例 |
|------|------|------|
| **分支选择** | 多路径叙事 | 《底特律：变人》 |
| **开放玩法** | 自由定义目标 | 《我的世界》 |
| **技能树** | 自定义发展路径 | RPG 技能系统 |
| **沙盒元素** | 自由探索创造 | 《塞尔达：旷野之息》 |

### 胜任感设计模式

| 模式 | 描述 | 示例 |
|------|------|------|
| **渐进难度** | 平滑学习曲线 | 大多数游戏 |
| **精通系统** | 高技能上限 | 《黑暗之魂》 |
| **可见进度** | 明确的成长 | RPG 属性、等级 |
| **挑战成就** | 认可高超技能 | 困难模式通关 |

### 关系感设计模式

| 模式 | 描述 | 示例 |
|------|------|------|
| **合作任务** | 共同目标 | MMO 团队副本 |
| **社交空间** | 社区聚集地 | 《动物森友会》 |
| **公会系统** | 长期社群归属 | MMO 公会 |
| **情感 NPC** | 深度角色关系 | 《质量效应》 |

## 评估问题

在评估游戏概念时问:

### 自主性
1. 玩家有多少真实选择?
2. 路径是否多样化?
3. 玩家能自定义体验吗?

### 胜任感
1. 成长是否可见且有意义?
2. 挑战是否公平?
3. 有足够的精通空间吗?

### 关系感
1. 是否支持玩家社交?
2. 合作内容是否深入?
3. NPC 关系是否有深度?
