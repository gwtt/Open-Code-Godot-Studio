## Key Features (Godot 4)

- GDScript 2.0 with improved typing
- Vulkan rendering (forward_plus, mobile, compatibility)
- Native 2D engine improvements
- Improved 3D rendering quality
- Better C# support
- GDExtension for C++/Rust
