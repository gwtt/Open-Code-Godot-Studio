# Puzzle Game Questionnaire

---

## Core Questions

1. **核心机制**: "核心谜题类型是什么？"
   - 匹配类（三消、连线）
   - 逻辑类（推理、排列）
   - 物理类（空间、碰撞）
   - 文字类（填字、单词）
   - 混合类型

2. **难度曲线**: "难度如何变化？"
   - 固定难度
   - 递进难度（关卡递增）
   - 自适应难度（根据表现调整）
   - 玩家选择难度

3. **提示系统**: "如何帮助玩家？"
   - 无提示
   - 有限提示（次数限制）
   - 无限提示
   - 渐进提示（逐步揭示）

4. **评分系统**: "如何评估玩家表现？"
   - 无评分
   - 时间评分
   - 步数评分
   - 综合评分（时间+步数+收集）

8. **社交元素**: "是否包含社交？"

   - 好友挑战
   - 分享功能
